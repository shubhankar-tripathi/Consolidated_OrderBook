
Intructions to setup and run the project::

1. Run tql in MySQL workbench to create orderbook table.
2. Change string parameter in create_engine and run Scratch.py in backend to populate MySQl.
3. Change string parameter in create_engine and run handler.py in wsapi to listen websocket connections.
4. Run ng serve --open while in the orderbook frontend folder.
5. Navigate and access realtime orderbook and snapshot of the order with restricted parameters.

Requirements::

1. python apis: sqlalchemy, sqlclient, tornado 
2. Angular CLI
